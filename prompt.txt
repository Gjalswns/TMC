🛠️ 작업 목표:
Next.js 13 이상 App Router를 사용하는 프로젝트에서
`/app/join/[code]/page.tsx` 파일이 서버 컴포넌트로 작동하도록 수정하고,
Supabase 데이터를 서버에서 안전하게 fetch 하도록 구성할 것.

🧩 요구 조건:

1. 클라이언트 컴포넌트 사용 ❌
   → 'use client' 제거
   → useEffect, useState 사용 ❌

2. `params.code`는 함수 매개변수로 전달되는 구조 유지
   → `export default async function JoinPage({ params }: { params: { code: string } })`

3. Supabase는 서버 컴포넌트에서 사용 가능해야 함
   → `lib/supabase-server.ts` 에 Supabase 클라이언트 생성 함수 새로 작성

4. `JoinGameWithCode` 컴포넌트는 그대로 유지
   → `game` prop으로 `{ id, title }` 전달
